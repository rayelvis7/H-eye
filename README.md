

#  Product Name: H-eye

##  What It Does
H-eye helps hotel owners track their rooms and bookings easily in one place.  
It replaces messy paper records with a simple digital system.
just like a digital hawk that monitors daily operation of the hotel 

##  Why It Matters
Many small and medium hotel owners still struggle with authenticating/juxtaposeing the records of the hotel activity which leads to 
- lost or incomplete records 
- difficulty in matching guest data with payments 
- difficulty during audits and inspection 

##  Who Itâ€™s For
- Small and medium hotel owners who want to monitor their hotel activity 
- External auditors who can get one- time access to review and verify hotel records  

##  Key Features
-  Guest check-in and check-out tracking  
-  Automatic billing reports- get summaries of expected income   
-  Dashboard showing room occupancy and income  
-  Cloud storage for data access anytime- all hotel data is stored offline for easy access anytime 

##  Technologies Used
- microcontroler: Arduino 
- Frontend: React native  + Tailwind CSS  
- Backend: Node.js + Express  
- Database: MongoDB  
- Hosting: Vercel + MongoDB Atlas
